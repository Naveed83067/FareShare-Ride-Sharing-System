# FareShare: A Ride-Sharing System

**FareShare** is a software engineering project dedicated to developing a localized, cross-platform ride-sharing application tailored for the Pakistani market. The system aims to provide a reliable, cost-effective alternative to existing international platforms by prioritizing fair pricing for riders and sustainable commission rates for drivers.

This repository serves as the central hub for all deliverables related to the FareShare project, developed as part of the Software Engineering course at Namal University.
